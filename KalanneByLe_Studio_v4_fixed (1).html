Workflow Photopea (miễn phí)

      

        1. Mở photopea.com → File → Open → chọn file PSD mockup

        2. Mở bảng Layers (cửa sổ bên phải)

        3. Tìm layer có icon Smart Object (icon hình vuông kép)

        4. Double-click vào Smart Object layer

        5. Cửa sổ mới mở ra → File → Place → chọn design của bạn

        6. Kéo/resize cho vừa → Enter để confirm

        7. Ctrl+S để save Smart Object → quay lại file PSD gốc

        8. File → Export As → JPEG / PNG → chọn 2000px

        9. Upload lên Etsy làm ảnh listing 🎉
      

    

  


  
  

    

      
Etsy Listing Photo Specs

      

        Kích thước tối thiểu: 2000 × 2000 px

        Tối ưu: 3000 × 2250 px (ratio 4:3)

        Format: JPG hoặc PNG

        Dung lượng: Max 10MB / ảnh

        Số ảnh: Tối đa 10, ít nhất 3


        5-ảnh checklist chuẩn:

        📷 1. Hero shot — design trên mockup chính

        📷 2. Lifestyle — phòng thực tế hoặc styled

        📷 3. Closeup — chi tiết chất lượng + text

        📷 4. Flat lay — toàn bộ sản phẩm

        📷 5. Info slide — size guide / file info
