Tải PSD về rồi → làm gì tiếp?
Dùng Photopea.com (miễn phí, không cài) để mở PSD và chèn design vào — xem tab "Cách chèn design" để có hướng dẫn từng bước.
Workflow: PSD mockup → ảnh Etsy hoàn chỉnh
1
Tải file PSD mockup từ Freepik / Unblast / Graphic Burger về máy.
File tải về thường là .zip — giải nén trước.
2
Mở Photopea.com trên browser → File → Open → chọn file PSD vừa tải.
3
Tìm Smart Object layer ở bảng Layers bên phải — thường tên là "Your Design Here", "Place Design", hoặc có icon hình vuông nhỏ góc dưới.
Nếu không thấy panel Layers: menu Window → Layers
4
Double-click vào layer Smart Object → một tab mới mở ra (đây là "canvas con").
Kéo file design của bạn (PNG/JPG từ Leonardo, Firefly...) thả vào tab này → resize cho vừa → Ctrl+S để lưu.
5
Quay lại tab chính — design của bạn đã được chèn vào mockup với đầy đủ shadow, texture, ánh sáng.
6
Export: File → Export As → JPG → chọn quality 90-95% → Save.
⚠ Export JPG (không phải PNG) — Etsy load nhanh hơn, file nhỏ hơn.
💡 Tips chọn mockup PSD tốt
• Ưu tiên file có chữ "Smart Object" hoặc "Editable" trong tên — chèn design chỉ mất 30 giây
• Tránh mockup quá nhiều text/branding của tác giả — mất thời gian xóa
• Resolution PSD nên từ 3000px trở lên — khi export JPG cho Etsy vẫn đủ sharp
• Chụp / screenshot preview mockup trên trang tải về trước — xem có hợp niche không rồi mới tải

📐 Etsy listing photo specs
• Kích thước tối thiểu: 2000px cạnh ngắn nhất
• Ratio tối ưu: 4:3 hoặc 5:4 — Etsy hiển thị đẹp nhất trên grid
• Format upload: JPG — file nhỏ hơn PNG, load nhanh hơn
• Không watermark trên listing photos — Etsy ToS không cho phép
• Số ảnh: 5-10 ảnh/listing — tối ưu conversion
🗂️ Checklist 5 ảnh cần có cho 1 listing
📸
Ảnh 1 — Hero mockup: Design trong khung đẹp, lifestyle background (phòng ngủ, bàn làm việc...). Đây là ảnh quyết định click-through rate.
🔍
Ảnh 2 — Closeup: Crop gần vào phần detail đẹp nhất — texture, nét chữ, màu sắc.
📦
Ảnh 3 — "What you get": Screenshot / preview các trang/file trong bundle. Quan trọng với digital products — khách cần biết họ nhận được gì.
📐
Ảnh 4 — Size reference: Design đặt cạnh vật quen (cốc cà phê, bàn tay, thước) để khách hình dung kích thước khi in.
🎁
Ảnh 5 — Flat lay / Lifestyle: Sản phẩm được in ra, đặt trên bàn với cây cảnh / cốc cà phê / sổ tay — tạo cảm giác "muốn sở hữu".
